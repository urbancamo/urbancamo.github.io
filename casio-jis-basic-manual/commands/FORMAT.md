## FORMAT

**[FX-870P/VX-4]**

### Purpose
Format a floppy disk.

### Format
```basic
FORMAT
```

### Explanation
1. Formats the floppy disk in the connected disk drive, preparing it for use.
2. All existing data on the disk is erased during formatting.
3. On the FX-870P/VX-4, no options are available for disk size.
4. On the FX-890P and Z-1, the `/6`, `/9`, and `/M` options are supported for
selecting different disk format sizes.
